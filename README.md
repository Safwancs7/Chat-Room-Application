# React Multi-Room Chat App

A real-time multi-room chat application built using **React.js** and **Firebase**.

## Features
- Google Authentication
- Multiple chat rooms
- Real-time messaging
- Typing indicator
- Modern clean UI
- Responsive layout

## Tech Stack
- React (Vite)
- Firebase Authentication
- Firestore Database
- CSS (Custom)

## How to run locally

```bash
npm install
npm run dev
